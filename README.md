# CheesKick Backend

Backend API for the CheesKick Flutter app.

## Current Status
- Phase 0 scaffold in place
- Custom user model wired in
- Docker, Redis, Postgres, CI, and linting foundations added

## Local Development
1. Copy `.env.example` to `.env`.
2. Install dependencies with Poetry.
3. Run migrations.
4. Start the app with `python manage.py runserver` or `docker compose up`.

## Notes
- The backend is designed around stats presentation and gamified experiences.
- It intentionally avoids real collectible ownership or NFT-like mechanics.
